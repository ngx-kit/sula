Sula is a starter-pack with simple style-guide, icons and a library of Angular components.

After download you will be able to modify all parts of the project by your requirements.

Structure

  • `projects/ui` - set of ui-components.
  • `projects/ui-docs` - representation of style-guide and ui-components.
  • `projects/app` - main application.

Usage

  • Download/fork the package.
  • Run `npm install`.
  • Run `npm run ui-docs:serve` and open `localhost:4100` to open UI-docs application.
  • Run `npm run app:serve` and open `localhost:4200` to open the main app.
