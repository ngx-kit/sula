Colors

Usage

Import params into .scss and use color variables.

Primary colors

Primary
$primaryColor
Main
$mainColor
White
$whiteColor
Light
$lightColor
Color Code Usage
$primaryColor Main actions like primary buttons, links, hover and active.
$mainColor Texts/icons, navigation background, borders and dividers.
$whiteColor Cards background, toolbar background, modals, forms and texts/icons.
$lightColor Main background of the application.

Primary color variations

Color Code Usage
$primaryColorDark1 Active state button and links color.
$primaryColorDark2 Hover state button and links color.
$primaryColor Main action and links.
$primaryColorLight1 Hover and active state.
$primaryColorLight2 Disabled state.

Main color variations

Color Code Usage
$mainColor First level navigation background and primary texts.
$mainColorLight1 Second level navigation background.
$mainColorLight2 Third level navigation background, hover state on navigation and scroll bars.
$mainColorLight3 Secondary texts.
$mainColorLight4 Disabled texts and borders.

Color Code Usage

$errorColorLight2 Disable state and background color in error status messages.
